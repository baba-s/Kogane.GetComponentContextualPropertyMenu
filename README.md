# Uni Get Component Contextual Property Menu

参照型のパラメータを右クリックした時のメニューに GetComponent を追加するエディタ拡張

## 使用例

```cs
using UnityEngine;

public class Test : MonoBehaviour
{
	public BoxCollider   bocCollider;
	public BoxCollider[] boxColliders;
}
```
